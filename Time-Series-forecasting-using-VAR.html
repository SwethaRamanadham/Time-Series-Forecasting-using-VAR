we have few ‘lag selection criteria’ like HQ, SIC, AIC and LR, etc. Generally, we choose the lag length for which the values of most of these lag length criteria are minimized. According to the more conservative SC(n) and HQ(n) criteria, the empirical optimal lag-order is 2.

Building the VAR model using VAR() by specifying the no of lags

var.2c <- VAR(Canada, p = 2, type = "const")

Testing causality to know if the real wage and productivity is causal to employment and unemployment.
